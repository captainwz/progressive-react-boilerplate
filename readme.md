## 渐进式React全家桶模板

[readme-EN.md](https://github.com/captainwz/progressive-react-boilerplate/blob/master/readme-EN.md)

渐进式React全家桶模板提供一系列React工程模板，结合各类概念进行讲解，由浅入深，达到快速掌握目的。
## 安装
```shell
npm install
```
## 版本
* react 16.2
* react-dom 16.2
* webpack 3.10.0
* babel-core 6.26.0
